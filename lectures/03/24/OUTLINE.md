* Code editors: Vim, Sublime, Visual Studio Code
* IDEs: Visual Studio, JetBrains, PyCharm, NetBeans
* `HTML`: HyperText Markup Language
	* version 5: most recent (HTML5)
	* works with `tags`
	* not a **programming** language per se
	* _structure_ to websites
	* `index.html`

* Content
	* `<!DOCTYPE html>`: tells the browser that this (current document) is an HTML file. open only
	* `<html></html>`: where all HTML "code" is stored
	* `<head></head>`: where the website's metadata are stored.
		* `<title></title>`: webpage's title
	* `<body></body>`: where everything visible is
		* `<p></p>`: makes a paragraph element
		* `<br>`: linebreak. goes to the next line. open only
		* `<span></span>`: used to style parts of a paragraph differently than the rest
		* `<h1></h1> <h2></h2> ... <h6></h6>`: heading elements. `<h1></h1>` is the largest, `<h6></h6>`, the smallest

	* Attributes are used to edit an element's behavior. they work via the general syntax: `attr_name="content"`
		* `style=""`: add CSS properties for an specific element (check CSS properties on the next outline)
